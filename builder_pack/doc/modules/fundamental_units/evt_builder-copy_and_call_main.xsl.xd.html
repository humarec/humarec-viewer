evt_builder-copy_and_call_main.xsl

EN: This stylesheet creates a copy of the document where the l element is closed (and later re-opened) whenever it is interrupted by a pb or lb element. For each page present in this copy the page template is applied (which is in evt_builder-main) IT: Crea una copia del documento nella quale il tag l viene chiuso (e poi ri-aperto) tutte le volte che è interrotto da un pb o un lb. Per ogni pagina presente in questa copia viene chiamato il template page (questo template si trova in evt_builder-main)
